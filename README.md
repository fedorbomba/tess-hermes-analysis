# Анализ астрофизических данных каталога TESS-HERMES DR1.0

Курсовой проект по анализу данных. Комплексное исследование астрофизических параметров звёзд методами машинного обучения.

## Описание проекта

Проект посвящён анализу набора данных TESS-HERMES Data Release 1.0 — каталога спектральных и фотометрических параметров звёзд, полученного в рамках миссии TESS (Transiting Exoplanet Survey Satellite) с использованием спектрографа HERMES.

**Источник данных:** https://archive.stsci.edu/prepds/tess-hermes/

## Содержание

- **Размер выборки:** 24 969 наблюдений × 41 признак
- **Категориальный признак:** класс звезды (hotdwarf, giant, cooldwarf, lmc, stflag_hermes)
- **Пропуски:** 161 820 значений (15,81%)

## Методы анализа

1. **Предобработка данных** — обработка пропусков, выбросов (IQR), one-hot encoding
2. **Корреляционный анализ** — матрица корреляций Пирсона
3. **Метод главных компонент (PCA)** — снижение размерности до 9 компонент (86,65% дисперсии)
4. **Кластеризация K-Means** — k=2, Silhouette Score = 0,216
5. **Линейная регрессия** — R² = 0,5037
